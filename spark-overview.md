# Apache Spark Overview
Apache Spark is an open-source library giving features to manage and analyse large volumes of data.

The library provides an easy to use set of classes to work with data. The library code itself does the heavy lifting behind managing large sets of data.

## Benefits
- __Easy to use__: Spark is a simple library for managing and analysing large data sets
- __Fast__: Spark processes data largely _in-memory_ and works on data _in parallel_
- __Unified__: Spark brings together data from different sources to treat in a consistent way

## Programming language support
Spark can be programmed using any combination of:

- Python
- SQL
- R
- Java
- Scala
